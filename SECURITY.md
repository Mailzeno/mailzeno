# Security Policy

MailZeno takes security seriously.

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue.

Instead, contact:

security@mailzeno.dev

Provide:

- Description of the issue
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 72 hours.

---

## Scope

This includes:

- API authentication
- Rate limiting logic
- Encryption mechanisms
- SMTP handling
- SDK behavior

Thank you for helping keep MailZeno secure.
